Rare Chain Nodes

Rare chain nodes are part of the infrastructure that sustains cryptocurrency such as bitcoin, Etherium and dash. Join the network by owning nodes for specific duration of time, rewards are given depending on activities carried out by your Nodes during a specific period of time.

  • Connect Wallet

    The connected wallet is used to fund Node activities. Rewards are also crdeited to the same wallet funding the node activity.

  • Choose Node

    Choose a Node and make sure to send the specified value to run the Node Network.

  • Earn Rewards

    Nodes run for specified duration, Rewards are automatically sent to the wallet address in contract the Node. NB: Early Reward withdrawals comes with significant gas fee charges.

How To Run Nodes On Rare Chain Network

Rare Chain Nodes Steps

Connect Wallet

The connected wallet is used to fund Node activities. Rewards are also crdeited to the same wallet funding the node activities.

Choose Node

Choose any Node network and make sure to send the specified value to activate the Node.

Earn Rewards

Nodes run for specified duration, Rewards are automatically sent to the wallet address in contract the Node. NB: Early Reward withdrawals comes with ignificant gas fee charges

Acquire Nodes

You can connect only via metamask and acquire available nodes. Always run node till specified duration to earn complete reward. Abruptly ending nodes will result to friction in node operations and will cost some gas fees.
